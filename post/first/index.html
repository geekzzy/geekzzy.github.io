0x01 创建网站

创建新站点(文件夹)

hugo new site web_blog
blog
├── archetypes  
├── config.toml //全局配置文件
├── content     //文章发布目录
├── data
├── layouts     
├── static
├── themes——|   //主题目录
            |——hugo-theme-dream———exampleSite      //模版目录
└── public      //编译发布目录

配置

cd web_blog
# 下载主题
git clone https://github.com/digitalcraftsman/hugo-agency-theme themes/hugo-theme-dream

# 本地启动
hugo server -t hugo-agency-theme --buildDrafts
